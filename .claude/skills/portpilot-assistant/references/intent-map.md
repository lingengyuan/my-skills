# Intent Map

## Chinese triggers

- 选端口: 帮我选个端口 / 给我一个可用端口
- 扫描端口: 扫描当前使用端口 / 看当前有哪些端口在监听
- 查占用: 看看 3000 谁在用 / 查看 3000 端口
- 释放端口: 释放 3000 / 清掉 3000
- 诊断: 扫描冲突 / 看端口打架
- 初始化: 初始化端口配置 / 扫描项目端口

## English triggers

- pick a port / find free port
- scan current ports / list listening ports
- who uses port 3000 / check port 3000
- free port 3000 / kill process on 3000
- run doctor / check conflicts
- init port config
